## Project Overview

This project focuses on building a loan classification pipeline to predict loan outcomes using machine learning.
The primary workflow lives in the notebook below:

- `Loan_classification.ipynb`

## Key Objectives

- Clean and prepare loan-related data
- Explore key patterns that influence loan decisions
- Train and evaluate classification models
- Compare performance using practical metrics
- Support data-driven credit risk decisions

## Quick Start

### 1) Create and activate a virtual environment

```powershell
python -m venv .venv
.\.venv\Scripts\Activate.ps1
```

### 2) Install core dependencies

```powershell
pip install jupyter pandas numpy matplotlib seaborn scikit-learn
```

### 3) Launch the notebook

```powershell
jupyter notebook
```

Then open:

- `Loan_classification.ipynb`

## Suggested Modeling Flow

1. Load and inspect the dataset
2. Handle missing values and outliers
3. Encode categorical features
4. Split into train and test sets
5. Train baseline and tuned classifiers
6. Evaluate with accuracy, precision, recall, F1, and ROC-AUC
7. Document findings and business implications
